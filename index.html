🔥 Connect Firebase

This is a one-time setup done by the site owner. Once saved, all visitors will share the same live database — no setup needed for parents.

1
Go to console.firebase.google.com → click Add project → name it (e.g. practice-champions) → click through to create it.
2
In the left sidebar click Build → Firestore Database → Create database → choose Start in test mode → pick any region → Done.
3
Click the ⚙️ gear → Project settings → scroll to Your apps → click the </> (Web) icon → register app → copy the firebaseConfig values below.
This is the most important one — it's in your Firebase URL

These values are safe to embed in a public site — Firebase security rules control access, not these keys.
